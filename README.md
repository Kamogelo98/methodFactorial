# methodFactorial

This is a function that takes in a number num and returns the product of all numbers from 1 up to and including num.
## Built With

* Visual Studio Code
* HTML
* JavaScript

## Tested With
Live Server

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites:
* Code Editor of your choice

### To clone the repo to your local repository:

``` bash
git clone https://github.com/Kamogelo98/methodFactorial.git
``` 

``` bash
cd methodFactorial
```

Open ``` index.html ``` in your preferred browser
